## ğŸš€ Getting Started

### Prerequisites
- **Node.js** (v16 or higher)
- **npm** or **yarn**

### Installation

```bash
# Clone the repository
git clone https://github.com/bhanukumardev/bhanu-portfolio.git

# Navigate to project directory
cd bhanu-portfolio

# Install dependencies
npm install
# or
yarn install

# Start development server

yarn dev
```

```bash

# or

# Preview production build
npm run preview
# or
```
